Usando Links Externos

Adicionar a tag A antes da frase que desejamos adicionar como hiperlink. parâmetro HREF é a url para qual a página será direcionada.

Usando links Externos

Para que o link externo não sobreponha o nosso site adicionamos os parâmetros TARGET="_BLANK" REL="EXTERNAL" do jeito que escrevi, só que minúsculas logo após a url, depois vem a frase e depois o fechamento da tag A.
