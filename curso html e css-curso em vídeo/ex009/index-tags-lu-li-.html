Listas de Definição

Usamos a tag DL para definir que se trata de uma lista de definição, DT para o termo a ser definido e a tag DD para a definição do termo.

HTML
Linguagem de marcação para a criação do conteúdo de um site.
CSS
Linguagem de marcação para criação de desing de um site.
JavaScript
Linguagem de programação para a criação da interatividade de um site.

Obs.: Nas listas definimos o tipo de ordem com o parâmetro TYPE dentro da tag OL ou UL e podemos definir de onde começa a ordem na OL com o parâmetro START após a info TYPE

Nas OL temos os TYPES: 1, A, a, I e i. E nas UL temos SQUARE, DISC e CIRCLE.
